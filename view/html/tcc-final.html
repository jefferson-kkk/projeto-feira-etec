Hardware

Análise de Hardware

Cada componente foi selecionado para maximizar a relação entre custo, eficiência e precisão. Abaixo, uma análise técnica detalhada de cada peça do hardware embarcado.

🧠
ESP32-WROOM-32
Unidade Microcontroladora
MCU Principal

Núcleo do AQM-32. SoC que integra processador dual-core, Wi-Fi e Bluetooth em um único chip — oferecendo capacidade de processamento robusta para leitura de sensores, controle de atuadores e comunicação sem fio simultaneamente.

Especificações
CPU Dual-Core Xtensa LX6 @ 240MHz
Conectividade Wi-Fi 802.11 b/g/n · BT 4.2
ADC 12-bit · 18 canais
Tensão 3.3V operação
📡
Sensor de Gás MQ-135
Unidade de Sensoriamento
Sensor Primário

Sensor semicondutor de óxido de metal (MOS) projetado para detectar NH₃, NOx, álcool, benzeno, fumaça e CO₂. A resistência interna varia com a concentração de gás, gerando um sinal analógico proporcional à poluição.

Especificações
Tipo Quimiorresistivo (MOS)
Gases NH₃, NOx, Álcool, CO₂
Tensão 5V operação
Saída Analógica (divisor de tensão)
🖥️
Display OLED 0.96"
Interface Homem-Máquina
IHM Principal

Display OLED monocromático de alta eficiência baseado no controlador SSD1306. Pixels pretos são desligados, reduzindo consumo. Controlado via I2C, exibe PPM, status e informações do sistema em tempo real.

Especificações
Tecnologia OLED Monocromático
Resolução 128 × 64 pixels
Controlador SSD1306
Interface I²C (SDA + SCL)
🚨
LEDs + Buzzer Ativo
Atuadores de Alerta
Output / Alerta

LEDs difusos (verde/amarelo/vermelho) + buzzer ativo 5V como sistema de alerta redundante. Resistores de 220Ω protegem os GPIOs. O buzzer possui oscilador interno — basta aplicar nível lógico alto para acionar.

Especificações
LEDs Verde · Amarelo · Vermelho (5mm)
Resistores 220Ω limitadores de corrente
Buzzer Ativo · ~2.5 kHz · 3.3–5V
Controle GPIO Digital (HIGH/LOW)

// AQM-32 · Sistema de Detecção de Gases
// ETEC Trajano Camargo — TCC 2026

#define PIN_MQ135   34
#define PIN_LED_G   25
#define PIN_LED_Y   26
#define PIN_LED_R   27
#define PIN_BUZZER  14

#define LIMIAR_ATENCAO  400
#define LIMIAR_PERIGO   700

Adafruit_SSD1306 display(128, 64);

void loop() {
  // 1. Leitura e conversão para PPM
  int raw = analogRead(PIN_MQ135);
  float ppm = convertToPPM(raw);

  // 2. Avaliação de status
  String status = "SEGURO";
  if (ppm >= LIMIAR_PERIGO)
    status = "PERIGO";
  else if (ppm >= LIMIAR_ATENCAO)
    status = "ATENCAO";

  // 3. Atualizar saídas
  atualizarLEDs(status);
  atualizarOLED(ppm, status);
  atualizarBuzzer(status);

  delay(500); // ciclo 2Hz
}

Aplicações e Futuro

🏭

Aplicações Práticas

O AQM-32 é ideal para monitoramento de qualidade do ar em ambientes fechados: residências, escritórios, laboratórios, garagens e oficinas. Alerta em tempo real sobre acúmulo de gases de combustão, vapores químicos e fumaça — protegendo pessoas e patrimônio.

🚀

Próximos Passos

A arquitetura modular do ESP32 permite expansões significativas: integração IoT via Wi-Fi para envio de dados à nuvem, aplicativo mobile com notificações push, data logging em cartão SD e adição de sensores especializados como MH-Z19 para CO₂ e BME680 para VOC.
